# 🌦️ WeatherInsight: Интерактивный анализ метеорологических данных

## Описание
**WeatherInsight** — это интерактивное веб-приложение на базе Streamlit для анализа и визуализации исторических метеорологических данных. Приложение предоставляет удобный интерфейс для метеорологов и аналитиков данных, позволяя исследовать погодные закономерности, сравнивать города и строить прогнозы.

### Основные возможности:
- **Интерактивные фильтры**: выбор городов, диапазона дат, фильтрация по дождливым дням
- **Производные столбцы**:
  - Категории температуры (❄️ Холодно, 🌱 Умеренно, ☀️ Тепло, 🔥 Жарко)
  - Категории осадков (💧 Без осадков, 🌧️ Небольшие, 🌊 Сильные)
  - Индекс комфортности (0-10) с категориями
- **Разведочный анализ**:
  - Гистограммы распределения
  - Box plots для сравнения городов
  - Корреляционная матрица
  - Анализ категориальных признаков
- **Временные ряды**:
  - Динамика показателей с группировкой по городам
  - Скользящее среднее (7 дней)
  - Сезонный анализ по месяцам
- **Прогнозирование**:
  - Прогноз на основе скользящего среднего
  - Настраиваемое окно и горизонт прогноза
  - Визуализация с историческими данными
- **Сравнение городов**:
  - Статистическая сводка
  - Сравнительные графики
  - Тепловая карта среднемесячных температур

## Установка зависимостей

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка
1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/weatherinsight.git
cd weatherinsight


### Создайте виртуальное окружение (рекомендуется):

python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate  # для Windows

### Установите зависимости:

pip install -r requirements.txt

### Инструкция по использованию
1.Боковая панель: настройте фильтры (города, период, дождливые дни)

2.Вкладка "Данные": просмотр отфильтрованных данных и базовой статистики

3.Вкладка "Разведочный анализ": исследование распределений и взаимосвязей

4.Вкладка "Временные ряды": анализ динамики и сезонности

5.Вкладка "Прогнозирование": построение прогнозов (для одного города)

6.Вкладка "Сравнение городов": сравнение нескольких городов
